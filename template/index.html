URL SHORTENER MICROSERVICE PROJET FROM FREECODECAMP WITH GOLANG

HOW TO USE THIS MICROSERVICE:

This api allow user to create a short version of a long url.

To do so you need to :

  • Option 1: entrer the long url you want to make short into the input field below.
  • Option 2: or simply put the long url next to: https://localhost:8080/new/.

    Exemple: http://localhost:8080/new/https://www.the_long_url.com. (Notice that this is not a valid url)

you will get a json data with the original url and it's short version.
